## How to install Particubes?

### iPhone & iPad

- Open this link on your device to join the beta program on Testflight: <a clicktracking=off href="https://testflight.apple.com/join/T1x6gC2L">testflight.apple.com</a>
- Follow the instructions to install Testflight (if you don't already have it) and Particubes!
- Testflight will tell you when updates are available.

### macOS

- Open this link to download the app: <a clicktracking=off href="https://particubes.com/download-mac-alpha">particubes.com/download-mac-alpha</a>
- Open the downloaded .dmg file
- Drag `Particubes.app` to `/Applications` (this is necessary if you want auto-updates to work as expected)
- Open Particubes and log in! (sorry, we just realized the password input field does not accept copy/paste... 😅)


### Windows & Android

Particubes is not ready for Windows & Android, but we're working on it. Stay tuned!

## What to test?

These are the things you can test in the current version:

- Install the app
- Log in using your password ({{ .Password }})
- Launch games from the grid. Note: we automatically generate one game per user for now, several entries may serve the exact same default script.
- Use "mine" filter to see the game that's been generated for you. I's not possible to create games yourself so far, your game is titled with your username.
- When playing **your** game, you can edit the game script from the pause menu (escape key on macOS, pause button on iOS). We would love to know if you thing the default script is easy to understand, even if you're a total beginner with code.
- Display the grid of items from the main menu. It's not possible to create nor edit items currently, but it will soon be unlocked.

Our main goal for now is to test inputs on mobile and desktop devices, connectivity, accounts, etc. We also want to see if our scripting environment is simple and appealing enough.

Please don't expect to find awesome game experiences in the list just yet. But hold on tight, we'll get there real soon!

If you plan to write your own scripts, have a look at the documentation, any kind of feedback is appreciated: <a clicktracking=off href="https://docs.particubes.com">docs.particubes.com</a>.

### Know issues / missing features

- Multiplayer sync is real bad. It should be much better in the next update.
- It's not possible to edit nor create game items
- It's not possible to edit nor create user ingame avatars
- It's not possible to create new games
- You can't upload a thumbnail for your game & game thumbnail placeholders don't look good 😆
